# Online Quiz (Java)

A simple console-based quiz application built using Java. This project asks multiple-choice questions, takes user input, and calculates the final score.

## Features

* Multiple-choice questions
* User input using Scanner
* Automatic answer checking
* Score calculation at the end

## Technologies Used

* Java
* OOP Basics
* Arrays & Loops

## Sample Output

```
1. Java is a ____?
A. Language
B. OS
C. Hardware
D. Browser
Enter Answer: A

Your Score: 3/3
```

## Learning Outcome

* Understanding of arrays and loops
* Handling user input
* Basic Java program structure
